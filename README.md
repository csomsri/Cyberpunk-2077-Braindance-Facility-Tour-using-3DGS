# Cyberpunk-2077-Braindance-Facility-Tour-using-3DGS
Inspried by CD Project Red's Cyberpunk 2077, we will replicate a Braindance to replay the memory of someone touring the facility.
## What is a Braindance?
[Cyberpunk 2077 Braindance Gameplay](https://www.youtube.com/watch?v=KXXGS3MGCro)

A Braindance is a feature in CD Project Red's Cyberpunk 2077 where you are replaying someones memory, feeling the sensations that the person's memory is feeling during the recording Braindance (BD). In this feature you are able to have two views, one where you are freely floating the 3D environment scanned and pickedup by the person's POV or replay the memory entirely in the POV. There are three layers in the BD that can be used in conjunction or filtered, Visual, Audio, and Thermal.
